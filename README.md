# My Portfolio Website

**A personal portfolio built with React and deployed on Azure Static Web Apps.**  
This project showcases my skills in **React, Azure, front-end development, and interactive UI components**.

## ?? Features
- **Responsive React app** with clean, modern design.
- **Interactive components**: PDF resume preview, download buttons, skill icons, and tooltips.
- **Dynamic skill section** with icons for C#, .NET, Azure, Node.js, React, Python, Git, GitHub, and AI.
- **Particle background effect** for visual appeal.
- **Azure Static Web Apps deployment** with CI/CD from GitHub.

## ?? Technologies Used
- **React** – Frontend framework  
- **React Bootstrap** – Layout and styling  
- **Azure Static Web Apps** – Hosting and deployment  
- **React Icons** – Skill icons and UI visuals  
- **React-PDF** – Preview and download PDF resume
